Library Terminal

First group project focusing on object oriented programming. This program successfully searhes through a library to check out and return books while logging changes to a text file so that the library may be updated for each use of the program.

Dice Roller

A dice rolling game designed to show proficiency with creating methods and using a list to log results.

Pig Latin Translator

A translator designed to convert an input to Pig Latin. This program sorts through individual words and rearrages them based on a set of rules based upon vowel placement while also excluding translation based on certain characters.
